# AI 荣耀闯关 🐵⚔️

> 游戏化 AI 可视化学习平台 · 面向职业院校云计算与数字媒体专业

以热门游戏 IP 为叙事主线，将人工智能核心知识点设计为六大闯关关卡。通关获得成就勋章并兑换学分。

## 🎮 六大关卡

| 关卡 | 主题 | AI 知识点 | 勋章 |
|:---:|------|---------|:---:|
| ① 灵石出世 | 花果山顶 | AI 定义 / 发展史 / 三大流派（符号/连接/行为主义） | 🪨 |
| ② 峡谷修炼 | 数据峡谷 | 机器学习训练流程 / 超参数调优 / 模型评估 | 🔰 |
| ③ 火眼金睛 | 火焰山 | CNN 卷积神经网络 / 计算机视觉 / AI 生成检测 | 🔥 |
| ④ 王者排位 | 王者殿堂 | 深度学习优化 / 策略梯度 / 模型竞技排名 | 👑 |
| ⑤ 七十二变 | 变化之森 | GAN / 扩散模型 / 潜空间插值 / 提示词工程 | 🪶 |
| ⑥ 团战时刻 | 团战平原 | 强化学习 / Q-Learning / 多智能体协同 | 💥 |

每关包含 3 个子步骤，共计 18 个进阶成就 + 10 个隐藏成就。

## 🎨 视觉风格

**水墨科创（东方科幻）** — 宣纸肌理、墨分五色、卷轴展开动画、Canvas 山水路径。

- Canvas 2D 水墨粒子背景（层峦叠嶂、云雾飘动、竹林摇曳）
- SVG 山峰剪影 + 印章收录动画
- 贝塞尔曲线山路连接（毛笔提按效果 + 苔点散布）

## ✨ 核心特性

- **AI 原理浮窗**：每个关卡内置「AI 原理」悬浮提示，讲解底层技术原理 + 游戏映射
- **影神册图鉴**：水墨图鉴收录 6 关神魔，3D 卡片翻转 + Canvas 手绘插图
- **隐藏成就**：10 个跨关卡隐藏成就（博览群书/火眼金睛/明察秋毫/炼丹宗师/速通达人…）
- **AI 知识总纲**：独立的知识汇总页，6 大分区，每个概念带 Wikipedia 链接
- **山水路线图**：首页贝塞尔曲线山路连接 6 关节点，如山水画底图
- **学分体系**：关卡通关 + 子步骤 + 隐藏成就 → 累计学分

## 🛠 技术栈

- **前端**: Vue 3 + Vite + Pinia + Vue Router（Composition API）
- **渲染**: Canvas 2D（水墨特效）+ GSAP（动画）
- **认证**: localStorage + SHA-256（服务抽象层，可切换后端）
- **部署**: Netlify（SPA）

## 🚀 快速开始

```bash
npm install
npm run dev      # 开发模式 → http://localhost:5173
npm run build    # 生产构建 → dist/
```

## 📁 项目结构

```
src/
├── views/              # 6 关卡视图 + 首页 + 影神册 + 知识总纲 + 教师后台
├── components/
│   ├── level1~6/       # 18 个关卡交互组件
│   └── ink/            # 通用墨水 UI 组件（AiPrincipleTip 等）
├── stores/             # Pinia 状态管理（authStore + gameStore）
├── services/           # 服务抽象层（authService + scoreService）
├── router/             # 路由守卫 + 关卡访问控制
└── assets/             # 静态资源
```
